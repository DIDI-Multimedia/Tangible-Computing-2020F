Examples

Creating an object and setting its properties, option 1 



    // double curly brackets is shorthand for an object
    let car = {} // This method creates a new object extending the prototype object passed as a parameter.
    car.type = BMW
    car.color = 'green'
    car.speed = 50

Creating an object and setting its properties, option 2 



      // same as option one 

      let car = {

        type: BMW,
        color: 'green',
        speed: 50
      
      }

      // or 

      let car = {type:'BMW',color:'green',speed:50}

Returning values from an object 



      // same as option one 

      console.log(car.type) // prints 'BMW' to console
      console.log(car['type']) // also, prints 'BMW' to console
