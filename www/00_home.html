DIDI Tangible Computing

Semester: Fall 2020

Instructor: Sayjel Vijay Patel




Course Description

'Tangible Computing' connects the ability to program a computer to our innate ability to grasp and manipulate objects. It examines how computers can behave as another material in a designer's toolkit. At the same time, students will gain insight into the field of computation and the evolving qualities of computers. Specifically, how to use their precision and extensibility to achieve otherwise unthinkable results. This course consists of three open-ended projects. The first project introduces the fundamental skills to program a computer. It touches on topics such as computer graphics and animation. The second project addresses the interface of contemporary software and hardware. The final project explores how computers interact with different sensory modalities and affordances.
